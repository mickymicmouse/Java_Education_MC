멀티 태스킹
두가지이상의 작업을 동시에 처리하는 것
멀티 프로세스
독립적으로 프로그램을 실행하고 여러가지 작업을 처리
멀티 스레드
한개의 프로그램을 실행하고 내부적으로 여러가지 작업을 처리

main스레드는 작업 스레드를 만들어 병렬로 코드를 실행

동시성(Concurrency) - 멀티 작업을 위해 하나의 코어에서 번갈아가면서 실행
병렬성(Parallelism) - 멀티 작업을 위해 멀티 코어에서 개별 스레드를 동시에 실행하는 성질

run Configuration -> arguments : 실행 프로그램에 필요한 매개변수를 전달할 수 있음

**** 동기화 메소드와 동기화 블록
읽는 것에 대해 동기화를 풀게 되면 여러가지 스레드가 접속 가능하다.
대표적인 모델 : consumer - producer 모델
